LDS (레이저 거리 측정) 방식#

LDS란?#

레이저를 360도 회전시키면서 거리를 측정하는 기술입니다. 마치 등대가 빛을 돌리는 것처럼 말이죠!

LDS의 장점#

  • 정확한 맵핑: 집 구조를 GPS 수준으로 정밀하게 파악
  • 어둠에 강함: 밤에도 문제없이 청소 가능
  • 프라이버시 보호: 영상을 촬영하지 않아 안전
  • 빠른 청소: 효율적인 경로로 시간 단축

LDS의 단점#

  • 작은 장애물 인식 부족: 신발끈이나 애완동물 배설물 감지 어려움
  • 높은 가격: 일반적으로 30만원 이상 가격대
  • 돌출된 센서: 침대나 소파 밑 청소 시 걸릴 수 있음

진화형: dToF (direct Time of Flight)#

2026년 삼성 비스포크에 적용된 차세대 거리 측정 기술이에요.

  • 빛을 쏘고 반사되어 돌아오는 시간으로 거리를 정밀 계산
  • 초당 10번 회전하며 반경 최대 8m를 360도 스캔
  • 기존 LDS보다 **감지 면적이 약 79% 증가 **(삼성 내부 테스트 기준)
  • LDS의 장점(어둠 작동, 프라이버시)은 그대로 유지하면서 정밀도 향상

진화형: RetractSense#

로보락 S9 시리즈에 적용된 기술로, LDS 센서가 자동으로 들어갔다 나오는 방식이에요.

  • 낮은 공간에서는 LDS를 본체 안으로 낮춰 통과
  • 열린 공간에서는 다시 올려서 정밀 맵핑
  • 기존 LDS의 “돌출 센서” 단점을 해결한 최신 기술
  • 더 얇은 디자인이 가능해져 침대/소파 밑 청소 범위 확대

이런 분에게 추천#

  • 넓은 집(30평 이상)에 거주하시는 분
  • 야간 청소를 자주 하시는 분
  • 프라이버시를 중요시하는 분
  • 가구 밑 청소가 중요하면 RetractSense 적용 모델 고려

카메라 방식#

카메라 네비게이션이란?#

카메라로 주변 환경을 촬영하여 공간을 인식하는 기술입니다. 과거에는 천장을 바라보는 카메라가 주류였지만, 2026년 현재는 전방을 촬영하는 AI 카메라가 대세예요.

구분 천장 카메라 AI 전방 카메라
촬영 방향 위쪽 (천장) 앞쪽 (전방)
공간 인식 천장 패턴으로 위치 파악 전방 사물 직접 인식
장애물 회피 기본적 AI 학습으로 세밀한 사물 인식
어둠 대응 불가 RGB + IR 조합으로 일부 대응
적용 모델 보급형 프리미엄 (복합센서와 함께 적용)

카메라 방식의 장점#

  • 우수한 장애물 회피: AI 학습으로 신발, 반려동물 배설물 등 세밀한 물체 인식
  • 얇은 디자인: 센서 돌출 없어 좁은 공간 진입 용이
  • 실시간 모니터링: 앱으로 청소 상황 확인 가능 (홈캠 기능)
  • 사물 인식 고도화: 최신 모델은 30종 이상 사물 구분 가능

카메라 방식의 단점#

  • 어둠에 취약: 조명이 없으면 인식률 급격히 저하
  • 프라이버시 우려: 집 내부 촬영으로 인한 보안 이슈
  • 천장 의존성 (천장 카메라): 천장이 복잡하거나 높으면 인식률 저하

이런 분에게 추천#

  • 작은 원룸이나 투룸에 거주하시는 분
  • 주로 낮에 청소하시는 분
  • 외출 시 반려동물 모니터링도 함께 원하는 분

복합센서 방식#

복합센서란?#

LDS + 카메라 + 각종 센서를 조합한 하이브리드 기술입니다. 마치 스위스 아미 나이프처럼 다양한 도구가 한 번에!

3D 구조광 센서란?#

복합센서에서 핵심적인 역할을 하는 기술이에요.

  • 적외선 패턴을 바닥에 투사해 장애물의 입체적 형상을 감지
  • LDS가 평면 거리만 측정하는 것과 달리, 3D 구조광은 높이·깊이까지 인식
  • 바닥에 놓인 신발, 전선, 반려동물 배설물 등 낮은 장애물 회피에 탁월
  • 정면 + 측면 구조광을 함께 사용하는 모델은 벽면 거리까지 정밀 측정

복합센서 조합 예시 (2026년 기준)#

프리미엄 구성:
LDS(공간 맵핑) + AI 카메라(사물 인식) + 3D 구조광(입체 감지) + 초음파(근접 거리)

고급 구성:
dToF(공간 맵핑) + AI 카메라(사물 인식) + 구조광(측면 감지)

중급 구성:
LDS(공간 맵핑) + 적외선(장애물 회피)

복합센서의 장점#

  • 최고의 성능: 각 기술의 장점만 결합한 올인원
  • 완벽한 장애물 회피: 작은 물체부터 큰 가구까지 정확히 인식
  • 환경 적응력: 밝음/어둠, 넓음/좁음 관계없이 일정한 성능
  • AI 학습: 청소할수록 점점 똑똑해지는 머신러닝 기능
  • 오염도 감지: 바닥의 오염 정도를 파악해 세척 강도 자동 조절

복합센서의 단점#

  • 높은 가격: 프리미엄 제품군에 주로 탑재
  • 복잡한 구조: 센서가 많아 A/S 비용이 상대적으로 높을 수 있음
  • 전력 소비: 여러 센서 동시 가동으로 배터리 사용 시간이 다소 짧음

이런 분에게 추천#

  • 반려동물을 키우시는 분 (배설물 회피 기능 필수)
  • 집에 장애물이 많은 분 (전선, 신발, 장난감 등)
  • 완벽한 청소를 원하시는 분
  • 어둠/밝음 관계없이 24시간 자동 청소를 원하는 분

구매 전 체크리스트#

1. 집 환경 확인하기#

  • 면적: 20평 이하 → 카메라/LDS 모두 무난, 30평 이상 → LDS/복합센서 유리
  • 조명: 어둠 청소 필요 → LDS/dToF/복합센서
  • 장애물: 반려동물, 작은 물건 많음 → AI 카메라 포함 모델 또는 복합센서

2. 우선순위 정하기#

  • 정밀한 맵핑이 우선 → LDS 또는 dToF
  • 장애물 회피가 우선 → AI 카메라 포함 모델
  • 둘 다 중요 → 복합센서

3. 사용 패턴 점검하기#

  • 야간 청소 자주 → LDS/dToF/복합센서 필수
  • 외출 시 자동 청소 → 모든 방식 가능
  • 실시간 모니터링/홈캠 → AI 카메라 포함 모델
